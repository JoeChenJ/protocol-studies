## 大纲

### Merge
- Altair / 轻客户端协议
- Merge / 提现
- 单槽最终性
- 秘密领导者选举

### Surge
- zk/op rollups
- KZG 仪式
- EIP-4844
- 数据可用性采样
- 跨 rollup 互操作性

### Scourge
- (非常) 简要概述 MEV
- ePBS / 包含列表 / MEV 燃烧
- 最大 EB，权益上限

### Verge
- Verkle 树
- SNARK 化一切：
  - Beacon 快速同步
  - Beacon 状态转换
  - Verkle 证明
  - EVM

### Purge
- EIP-4444
- 协议简化

### Splurge
- EIP-1559 终极目标
- 账户抽象
- 深度密码学
